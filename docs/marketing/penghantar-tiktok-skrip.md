## Cara guna (rakam jadi video)
1. Buka `penghantar-tiktok-demo.html` dalam browser (Chrome/Edge), tekan **F11** untuk full screen.
2. Animasi auto-main dan ulang sendiri. Tekan skrin untuk **main semula**.
3. Rakam guna salah satu cara:
   - **Screen record** terus (telefon/PC), pastikan rakam kawasan 9:16 di tengah.
   - Atau guna **OBS Studio**: set canvas 1080×1920, tangkap window browser.
4. Tindih lagu trending TikTok + voiceover (skrip di bawah).
